About Myself

Hi, my name is Kevin Tang. I'm a recent graduate from the University of Illinois at Chicago, where I obtained a Bachelor's Degree in Computer Science with a Software Engineering concentration. Most of my education consisted of OOP in Java and C#, relational databases using SQL, agile development methodologies, algorithms, and some C/C++.

On my journey in exploring the many facets of the tech industry, I kicked off my career working as a Software Development Engineer in Test Consultant, picking up a wide variety of tools, methodologies, and experiences.

As an SDET, I've had the opportunity to play an active role in the software development life-cycle at the a client company, including spearheading the test planning, documentation, and execution processes across 5 different projects, end to end. I was able to demonstrate my C# skills through UI automation tests utilizing Selenium WebDriver and significantly cut down the time spent in manual testing phases by creating tools for manual testers that made mock data generation, test monitoring, and test report generation. It is throughout my experience in software testing that I realized that my passion lies in developing well written code and creating solutions to make complex problems simple. I am now pursuing a new opportunity as a Software Engineer.

Marvel vs. Capcom: Infinite Frame Data
Web development was something that I had overlooked throughout my college career, as most of my focus was on Java and C# development. With plenty of free time on my hands, I'm looking to strengthen my HTML, CSS, and JavaScript skills further, as well as explore some popular JS libraries and frameworks, such as jQuery and AngularJS.

This application is used to display statistical information and frame data about a character's move-set and calculates optimal options in response to each move based on the user's character selection. The application is set up on an Apache web server ran on an AWS EC2 instance, and communicates with a RESTful PHP web service that pulls data from a MySQL database ran on AWS RDS.
